# Agent Memory 架构简图

## 整体分层
1. 接入层
   - 用户输入
   - 历史记忆加载

2. 记忆管理层
   - 短期记忆：对话历史
   - 中期记忆：摘要/压缩
   - 长期记忆：RAG 外部知识库

3. 调度层
   - 决定何时读取记忆
   - 决定何时写入记忆
   - 决定何时遗忘/裁剪

4. 执行层
   - LLM 思考与决策
   - 工具调用
   - 结果返回并更新记忆

## 与 Agent 其他模块关系
- Memory ↔ RAG：长期记忆检索增强
- Memory ↔ Planner：基于历史做连续规划
- Memory ↔ Tools：工具调用结果写入记忆，形成闭环

## 工程关注点
- 线程安全 / 异步安全（asyncio 场景）
- 记忆持久化与过期策略
- 检索性能与召回率
- 可观测性：日志、trace、回放
